Coding HTML and CSS

HTML and CSS are languages that are used to design websites. They are so usefull, that they have been around for over 20 years! And in the IT world, technology changes so fast that things can become outdated within a year. So it is very impressive these languages have been in use for such a long time.

This booth will give you a guide to creating your very own webpage, that can tell a story, show your art, or just make people laugh. ☺

Requirements

You don't need much to start coding, just open your favorite text editor.

If you are using a windows computer, click the windows key on the bottom left hand corner of your screen, then type 'notepad' and hit enter.

If you are using a MAC computer, navigate to applications, then click 'TextEdit'.

* Ask a mentor at the booth if you need help!

Next, open up your favorite web-browser. Like Firefox, Google Chrome, or Safari.

How a Webpage Works

The web-browser grabs the HTML file for the website, and any other required files.

The HTML file tells the web-browser what text to display, and any other files the browser may need to display that content.

For now we will only have one HTML file.

Making the HTML File

After opening our favorite text-editor, we will save this file as an HTML file on our Desktop.

To do this in Windows, click 'File' on the top left corner of notepad, then click 'Save As...'

Now, navigate to your desktop folder, and save it as 'index.html'.

Viewing the Web-Page

Open this file in your web-browser, by right-clicking the 'index.html' file on your desktop.

* If it does not work, try right-clicking the 'index.html' file, and selecting 'Open With', then choose the web-browser.

When we p our index file, we will be able to view it here. As you can see, the webpage is empty! This is because we have not written anyhting yet, so lets go back to the notepad file we have open.

Basic HTML

Now lets start coding!

The first thing we will want to do is decide what our webpage will be about. Will it be about robots fighting? How about Ninjas? The only limit is your imagination!

This example is going to be about cats, but feel free to replace it with whatever you want.

<h1> - this is a header tag, it is used to make a title. Go ahead and type '<h3>Cats!</h3>' into your index file. Then Save it.

After saving, go back to your browser window and click the refresh button, or hit 'F5' on your keyboard. You should see the word 'Cats!' in your browser.

Add More Content

Lets start writing about your subject, wether it be cats or ninjas.

<p> - this is the paragraph tag, it is used to make small entries of text within a webpage. Go ahead and type your text like this, right below your title, like this;

<h1>Cats!</h1>
<p>Cats are the dominant domesticated animal of America, and are very fluffy.</p>
